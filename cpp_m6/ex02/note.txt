a new concept : runtime type identification (RTTI) in polymorphism
RTTI = the ability for C++ to know the real derived type at runtime
-> to enable the RTTI, a class must have at least one virtual function
-> so here, Base* p = new A(), with RTTI, the object is actually A
                                without RTTI, p is Base*, we lose information


dynamic_cast<real_type_ptr*>(object)
ex: Base* p = new A();
A* a = dynamic_cast<A*>(p); -> OK
B* b = dynamic_cast<B*>(p); -> not OK return NULL
-> it means that is the object (p) find its real type, it is ok, if else it return NULL
