CANONICAL FORM:
    - default constructor : example();
        -> create a brand new object in a valid state
        -> initialize all members
    - copy constructor : example(const example& other);
        -> create a new object that is a clone of another
        -> not assignment
    - copy assignment operator : example& operator=(const example& other);
        -> called when existing object is replaced by another
        -> this object already exists, make it become equal to another
    - destructor : ~example();
        -> called when the object dies
        -> clean up and release resources
