        -> the method to call depends on the actual object
            not on the pointer type.
        -> use just "virtual" in the base class declaration
            and not required in other class override.(make virtual once)
        -> if we use "new" to create object, we need to make the destructor virtual too,
            for delete. Because delete looks at the pointer type, not the actual object type,
            unless the destructor is virtual.
